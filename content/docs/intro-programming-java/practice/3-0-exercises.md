Scenario-based problems covering primitive data types, arithmetic operators, and user input and output with `Scanner`. Each program should collect input from the user at runtime — avoid hardcoding values unless the problem says otherwise.

---

## Problem 1 — Grocery Bill

A customer at a grocery store is buying three items. They want to know the total cost before they get to the checkout.

Write a program that asks the user to enter the price of three items and prints the total cost.

---

## Problem 2 — Rectangle Dimensions

A contractor needs a quick tool to calculate the area and perimeter of a rectangular room given its width and length.

Write a program that asks the user for the width and length of the room in metres and prints both the area and the perimeter.

---

## Problem 3 — Speed Calculator

A driver wants to know their average speed for a trip. Speed is calculated as distance divided by time.

Write a program that asks the user for the distance travelled in kilometres and the time taken in hours, then prints their average speed in km/h.

---

## Problem 4 — Coin Counter

A student empties their piggy bank and counts their coins. They have toonies ($2.00), loonies ($1.00), quarters ($0.25), dimes ($0.10), and nickels ($0.05).

Write a program that asks the user how many of each coin they have and prints the total value in dollars.

---

## Problem 5 — Age in Days

A curious person wants to know approximately how many days old they are.

Write a program that asks the user for their age in years and prints an approximate number of days they have been alive. Use 365 days per year.

---

## Problem 6 — Splitting a Bill

A group of friends wants to split a restaurant bill evenly. They know the total amount and how many people are splitting it.

Write a program that asks the user for the total bill and the number of people, then prints how much each person owes.

---

## Problem 7 — Celsius to Fahrenheit

A weather app needs to display temperatures in both Celsius and Fahrenheit. The conversion formula is: `F = (C × 9.0 / 5) + 32`.

Write a program that asks the user to enter a temperature in Celsius and prints the equivalent in Fahrenheit.

---

## Problem 8 — Savings Goal

A student wants to know how many weeks it will take to save up for something they want to buy. They know how much they have already saved and how much they set aside each week.

Write a program that asks the user for their current savings, their weekly savings amount, and their goal amount, then prints the number of weeks until they reach their goal.

---

## Problem 9 — Formatted Receipt

A small coffee shop needs a basic receipt printed to the console. The receipt should display the item name, quantity, unit price, and calculated total — formatted neatly so the values line up.

Write a program that asks the user for an item name, a quantity, and a unit price, then prints a formatted receipt.

**Example output:**
```
-----------------------------
Item:     Latte
Quantity: 3
Price:    $4.50
Total:    $13.50
-----------------------------
```

---

## Problem 10 — Data Type Decisions

A developer is designing a program to store information about a used car listing. The listing needs to hold: the make, the model, the year, the odometer reading in kilometres, the asking price, and whether the car is certified pre-owned.

Write a program that declares a variable for each field using the most appropriate data type, assigns realistic values, and prints the full listing to the console with labels.

For each variable, be prepared to explain why you chose that data type.
