Loops allow a program to repeat actions without writing the same code over and over. This topic covers the three loop types in Java and how to control them.

## Learning Objectives

- Develop loop-based solutions using for, while, and do-while
- Control loop execution using break and continue
- Analyze loop efficiency and logic
