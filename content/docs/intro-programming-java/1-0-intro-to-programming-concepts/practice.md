# Practice Set 1: Introduction to Programming Concepts

These labs are built for beginners. Keep your answers short, use plain language, and focus on how you would think through the problem before worrying about code.

## Lab 1: Input, Process, Output

Pick one scenario and break it into IPO:

1. A student uses a vending machine
2. A user enters a tip at a restaurant
3. A calculator adds two numbers

For your chosen scenario, write:

- **Input**: What information goes in?
- **Process**: What happens to that information?
- **Output**: What comes out at the end?

## Lab 2: Turning a Real-World Task Into Steps

Imagine you are helping a friend make a simple app that tells them whether they have enough money for lunch.

Write 4 to 6 steps that the program would follow.

Use this format:

1. Ask for the amount of money the person has
2. Ask for the cost of lunch
3. Compare the two values
4. Show a message based on the result

## Lab 3: Spot the Logic

Read each situation and decide if it needs a clear step-by-step process:

- Making a peanut butter sandwich
- Choosing a movie based on genre
- Guessing a random number

For each one, write one sentence explaining why a computer would need organized logic to solve it.


## Lab 4: Input or Output?

Label each item as **input**, **process**, or **output**:

- A user types their name
- The computer adds two numbers
- The screen shows a message
- A person presses a button
- The program checks whether a number is bigger than 10

## Check Yourself

When you finish, ask:

- Did I identify the input clearly?
- Did I explain the process in order?
- Did I describe the output in a simple way?

If you can answer yes to all three, you are thinking like a programmer.
