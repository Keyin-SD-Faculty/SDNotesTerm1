Conditional logic lets a program make decisions. This topic covers the structures Java provides for branching program flow based on conditions.

## Learning Objectives

- Implement decision-making structures (if, if-else, switch)
- Apply Boolean logic and comparison operators
- Analyze program flow using conditional structures
