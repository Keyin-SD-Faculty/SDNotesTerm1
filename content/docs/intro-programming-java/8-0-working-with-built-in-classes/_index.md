Java's standard library includes powerful built-in classes that handle common tasks. This topic covers the most commonly used ones: String, Math, wrapper classes, and date and time tools.

## Learning Objectives

- Use String, Math, and wrapper classes
- Manipulate text and numeric data
- Apply date and time functionality
