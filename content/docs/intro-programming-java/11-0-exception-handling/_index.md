Things go wrong at runtime — users enter bad data, files don't exist, connections fail. Exception handling lets your program respond to these situations gracefully instead of crashing.

## Learning Objectives

- Explain exception handling concepts
- Implement try-catch-finally structures
- Diagnose and handle runtime errors
