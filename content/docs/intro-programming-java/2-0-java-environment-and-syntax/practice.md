# Practice Set 2: Java Environment and Syntax

This set introduces the tools and rules you need before you start writing Java code. Keep it simple and focus on understanding what each part does.

## Lab 1: Match the Java Tools

Write the correct term next to each description:

1. The part that runs Java bytecode
2. The package that includes the tools needed to build Java programs
3. The environment that lets you run Java programs

Terms to use:

- JVM
- JRE
- JDK

## Lab 2: Your First Hello World

Read this program and answer the questions below:

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

Questions:

1. What is the name of the class?
2. Which line starts the program?
3. What text appears on the screen when it runs?

## Lab 3: Syntax Check

Look at these Java rules and explain them in your own words:

- Every statement ends with a semicolon
- Curly braces show where a block starts and ends
- Java is case-sensitive
- Comments help explain code without changing how it runs

## Lab 4: Compile or Run?

Decide whether each command is used to compile or run Java:

- `javac HelloWorld.java`
- `java HelloWorld`

Then write one sentence explaining the difference between the two commands.

## Lab 5: Comments in Code

Write a single-line comment and a multi-line comment for a program that prints a welcome message.

Your comments should explain:

- what the program is for
- what part of the code is important

## Lab 6: Fix the Syntax

Read this code and list three things that look wrong:

```java
public class helloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!")
    }
}
```

Hint: think about case sensitivity, punctuation, and where statements end.

## Check Yourself

You are ready to move on if you can:

- name JVM, JRE, and JDK correctly
- recognize the structure of a simple Java program
- explain the difference between compiling and running
- identify comments and basic syntax rules
